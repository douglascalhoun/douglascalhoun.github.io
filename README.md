# House Jams 🎸

An interactive guitar chord and lyrics viewer for musicians.

## Features

- **60+ Songs** - Classic folk, country, and rock songs with chords
- **Chord Toggle** - Show/hide chord notations with a single click
- **Auto-Scroll** - Hands-free scrolling with 10-second countdown
- **High-Contrast Mode** - Easy-to-read dark theme for low-light playing
- **Keyboard Shortcuts** - Navigate songs with `&` (previous) and `(` (next) keys
- **Persistent Settings** - Remembers your last song and preferences

## Usage

Simply open the page and start playing! Use the controls at the bottom:

- **songs** - Browse the song list
- **chords** - Toggle chord visibility
- **colors** - Switch to high-contrast mode
- **scroll** - Enable auto-scrolling
- **+ add** - Add your own songs (coming soon)
